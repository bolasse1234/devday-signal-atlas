# Signal Atlas

Signal Atlas is a DevDay 2026 ticket-contest submission built with GPT-5.5 and Image Gen.

It is a self-contained Vite/React browser studio that turns a rough AI idea into:

- a demo arc
- an Image Gen prompt
- a contest reply
- a downloadable poster

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
